Privacy Policy

Last updated: June 2026

Overview

Element Hider Pro allows you to hide, blur, randomize text, or replace words on web pages you visit. It stores your settings and rules locally in your browser.

Data Collection

Element Hider Pro does not collect, transmit, or share any personal data. All data stays entirely on your device.

Data Storage and Security

All extension data is stored locally using chrome.storage.local (or browser.storage.local on Firefox). This includes:

  • Your hide/blur/randomize rules per page
  • Custom word replacement rules per site
  • Your preferences (theme, blur intensity, hide strategy)
  • The list of sites where the extension is disabled

No data is sent to any external server.

Data Retention

Locally stored data persists until the user uninstalls the extension.

Third-Party Services

  • Favicons: The dashboard loads site favicons from Google's favicon service (www.google.com/s2/favicons). This is a standard, anonymous HTTP request that does not identify you personally.
  • Export/Import: When you export rules, a JSON file is downloaded to your device. When you import, you choose which file to load. No data leaves your browser.
